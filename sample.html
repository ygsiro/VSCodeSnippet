"include-guard": {
  "prefix": "#hdr",
  "body": [
    "#ifndef ${1:${TM_FILENAME/(?:(\\w+)(\\.)?)/${1:/upcase}${2:+_}/gi}}",
    "#define $1",
    "$0",
    "#endif // $1"
  ],
  "description": "インクルードガード"
}

ファイル名を大文字に変換しつつ.を_に変換する正規表現をプレースホルダーの初期値に設定している。 #ifndefの行でしか正規表現は使用していませんが、 タブストップのリンク機能により#defineおよび#endifの行でも正規表現によって変換されたファイル名が挿入される。
